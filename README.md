# Решение задач из курса весеннего практикума для студентов 2-го курса 1 потока ВМК МГУ
**Язык программирования: C++**

Привет! Этот репозиторий создан для того, чтобы помочь вам сориентироваться в весеннем практикуме. Здесь собрана основная информация о том, как устроен курс и как не потерять баллы

## Структура курса:
*Получение условий и сдача кода проходит на платформе Ejudge.*

- 10 машзальных контестов из *(в среднем)* 5 небольших заданий с возможностью дорешивания дома, направленные на реализацию алгоритмов или отработку навыков, полученных на семинаре

- 8 лекторских задач, дающих дополнительные баллы на коллоквиуме по курсу "Системы программирования"

*Важно учитывать: варианты заданий для каждого студента одинаковые, посылки проверяют на антиплагиат (особенно касается лекторских задач)*

## Система оценивания:
- Каждое верное решение задачи (=прохождение посылкой всех скрытых тестов) во время "машзальных" семинаров дает дополнительные **+100 баллов** вне зависимости от количества предыдущих "штрафных" посылок
- Каждое неудачное решение (=не прошедшее все скрытые тесты) отнимает **5 баллов** от итогового балла за задачу. Однако, как бы много раз вы ни ошиблись, за решенную задачу вы получите **не менее 50 баллов**
- Сдача задачи после дедлайна постепенно снижает количество получаемых баллов вплоть до **0**. Старайтесь закрывать контесты вовремя!
  
## Структура репозитория:

Задания из курса разбиты по папкам согласно следующему шаблону:
```text
    ├── taskN
    │ ├── taskN_1.cpp
    │ │   ...
    │ └── taskN_M.cpp
    │
    ├── SP_contests
    │ ├── 1_<task_name>.cpp
    │ │   ...
    │ └── 8_<task_name>.cpp
```
где N - номер контеста, M - номер задачи в контесте N
